# WS2812B-Clock
Orange Pi addressable LED library.

Tested boards:

    Orange Pi lite2
  
  
To run test example:
 ```
git clone https://github.com/k773/WS2812B-Clock
cd WS2812B-Clock/
mkdir build
cd build
cmake ..
make
./W2812Clock
```
Make sure you have spi enabled and your leds are properly connected to spi pin.
